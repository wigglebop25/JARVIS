### JARVIS Frontend Dashboard Requirements
#### Device Information
- Display CPU usage, RAM usage, Storage usage, and Network status for each linked device.
- Provide real-time updates on device status and resource usage.
#### Automation Routines
- Allow users to create, edit, and delete automation routines.
- Routines can be triggered based on time, device status, or user commands.
- Provide a user-friendly interface for managing routines, including a visual representation of the routine flow.
#### Device Management (Exclude from MVP)
- Enable users to link new devices to the system.
- Display the number of devices currently online and offline.
- Provide the ability to wake devices using Wake on LAN.
#### User Interface
- Design a clean and intuitive dashboard using React.
- Ensure the dashboard is responsive and works well on various screen sizes.
- Implement real-time updates using Websockets to reflect changes in device status and resource usage immediately.
